Mall Management System

Developed Role-Based Access Control using JWT Authentication to secure Admin and Store Owner operations in a Mall Management System.

Built Admin and Store Dashboards to manage contracts, users, invoices, stores, products, sales, and notifications, enhancing operational efficiency.

Implemented Data Access Layers with Entity Framework for seamless integration and ADO.NET with stored procedures for performance optimization.

Designed and Developed Frontend using Razor Views, jQuery, and Kendo for listing.

Optimized Security and Scalability by integrating role-based security mechanisms and leveraging SQL Server for robust data management.

  • Entity Framework
  • ADO.NET
  • SQL Server
